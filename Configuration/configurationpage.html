YouTube Feed Settings

How to export your YouTube cookies

  1. Install the Get cookies.txt LOCALLY extension from the Firefox Add-ons store.
  2. Make sure you are logged into YouTube in Firefox.
  3. Click the extension icon in the toolbar while on youtube.com.
  4. Select "Export" → "Current Site" to download youtube.com_cookies.txt.
  5. Copy the file to your Jellyfin server:
    scp ~/Downloads/youtube.com_cookies.txt yourserver:/home/youruser/youtube-cookies.txt
  6. Enter that path in the Cookies File Path field below and save.

Note: Cookies expire periodically. If the feed stops updating, re-export and replace the file.
